Objective: (CAD translation service) Convert the CAD file from one format to a different one. It supports the most CAD formats.

User workflow

  1. User inputs the URL of the translator to the User Agent (Browser). i.e., http://3dtrans.com

  2. User select the source file storage provider like A360, Box.com or QQ zone.

  3. User Agent navigate the user to the authorization page of the storage provider.

  4. User input the account and password and grant the authorization.

  5. User Agent navigate the user to the translator page and all the supported files are displayed in a dialog.

  6. User select the source file.

  7. User select one or more destination file formats with the check box.

  8. User click the start button to start the translation.

  9. Translator service translate the file and notify the user once the work is completed.

  10. User receives the notification and s/he can access the destination files from the storage provider.

Translation service workflow

  1. Download the source file from the storage server.

  2. Service calculates translation paths from the source to the destinations.

  3. Start the corresponding CAD products to translate the source file to the destination formats.

  4. Upload the destination files to the storage server.

  5. Send notification to the user that the translation is completed.
